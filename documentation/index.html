Installation

  1. Via Wodpress Admin: Go to Appearance-> Install Themes -> Upload and choose lawyers.zip file ((Make sure you are not uploading the whole zip file that you've downloaded (including dummy content, psd file, documentation) but only the lawyers.zip file). Click Install Now.
  2. Via FTP: Upload lawyers folder into wp-content/themes folder.
  3.  After you've installed the theme make sure to activate it.

Install dummy content

If you want your website look like in my live demo, than customize it as you want, follow these steps:

  •     Click Tools > Import
  •     Click on WordPress
  •     Install the Importer by clicking "Install Now" and then "Activate Plugin & Run Importer"
  •     Upload the xml file from demo xml folder: "lawyers-demo-xml.xml".
  •     Now assign your posts to an existing user, check the "Download and import file attachments" checkbox and click "Submit".

Now follow this steps:

  • Go to Appearance > Menus, select "Primary Menu" at the bottom of the page and click Save.

  • Go to Settings > Reading and select Front page displays "A static page".
    Front page: "Home".
    Posts page: "Blog".

After you've installed the theme, you can customize it in the Appearance -> Theme Options.

If you didnt install XML file like in preview, then go STEP BY STEP....

 

Setup Menu

Note: Before creating the menu items make sure you've created some pages.

1. Go to Appearance - > Menus tab.

2. Add new menu, enter Menu Name.

3. Add Menu Items. Select the pages you've just created and add them to the menu.

4. Drag & drop the menu items as you like. Check the image below. Than save the menu.

5. !IMPORTANT: make sure the Theme Locations option is set to Primary Menu.

Pages General

Custom Page Templates

First go to Settings -> Permalinks. Then you can see Common Settings and choose Post name. This is for better SEO permalinks.

The theme comes with the following custom Page Templates:

  • Default Template - The default page.
  • Home Template - A fully widgetized page.
  • About Template - A page to display the about info.
  • Practice Areas Template - A page to display the practice areas.
  • Team Template - A page to display the team.
  • Case Result 1 Col Template - A page to display the case results (1 columns).
  • Case Result 2 Col Template - A page to display the case results (2 columns).
  • Testimonials Template - A page to display testimonials.
  • Contact Template - A page to display contact form and info.

How to use a template:

  • Click Pages > Add New.
  • Look for Page Attributes on the right side and click on the "Template" drop-down menu.
  • Choose a page template.
  • Click the Publish button above.
  • Now you can add your new page to your Menu.

Page Customization Meta Box:

Each page has a customization meta box where you can customize the page title, tagline text and add top background image.

Setup Home

Setting up the Homepage:

  1. Click Pages > Add New.
  2. Look for Page Attributes on the right side and click on the Template drop-down menu. Select the "Home" template.
  3. Click Publish.
  4. Now you can add the new page to your menu.
  5. Now click Settings > Reading and select Front page displays a static page. Select the page you just created as your Front page and click Save Changes.
  6. The Homepage is fully widgetized. Simply go to Appearance > Widgets and add only widgets that contains only with MT - Home title.

Homepage Slider:

To configure the homepage slider go to Appearance -> Theme Options -> Home Top Slideshow. 

Note: The slider images have a dimension of 1600x700 px. You can experiment with other sizes too.

Setup Blog

Setting up the Blog Page:

  1. Click Pages > Add New.
  2. Look for Page Attributes on the right side and click on the Template drop-down menu. Select the "Default Template".
  3. Click Publish.
  4. Now you can add the new page to your menu .
  5. Now click Settings > Reading and select Front page displays a static page. Select the page you just created as your Posts page and click Save Changes.
  6. Click Posts > Add New to create a news post. Make sure when you add a Featured Image for a post to have at least 800x360 px to look like my demo.
Note: The top image for each post that you create is the same as the Blog Page top image. So you just have to add only the Blog Page image.

Setup About

Setting up the About Page:

  1. Click Pages > Add New.
  2. Look for Page Attributes on the right side and click on the Template drop-down menu. Select the "About" template.
  3. Click Publish.
  4. Go to Appearance -> Theme Options -> About Page to add sections.

Setup Practice Areas

Setting up the Practice Areas Page:

  1. Click Pages > Add New.
  2. Look for Page Attributes on the right side and click on the Template drop-down menu. Select the "Practice Areas" template.
  3. Click Publish.
  4. Click Practice Areas > Add Practice Item
  6. Add text and icon found in the Item Customization meta box

  8. Now you can choose how to display the Item Single Page: via Modal Window or via Normal Page. For that you'll have to go to Appearance > Theme Options > General Settings > Practice Areas - Choose Single Item Display . Also make sure the Practice Area Single Item - Parent Page Select ( below option ) is set to the Pracice Areas template page.

Setup Case Results

Setting up the Case Results Page:

  1. Click Pages > Add New.
  2. Look for Page Attributes on the right side and click on the Template drop-down menu. Select the "Case Results 1 Col" template or "Case Results 2 Col" template.
  3. Click Publish.
  4. Click Case Result > Add Case Result Item
  5. Complete the fields and add Featured Image. Make sure the image is at least 540x300 px.

Setup Contact

Setting up the Contact Page:

  1. Click Pages > Add New.
  2. Look for Page Attributes on the right side and click on the Template drop-down menu. Select the "Contact" template.
  3. Click Publish.
  4. In the Theme Options Panel go to Contact Page and edit the fields
  5. Make sure the Your Email Address field has a valid email. At this email you'll receive your messages.
  7. The Email Right Side field is just for visual display in the contact page and top right header.

Footer

Go to Appearance > Theme Options > Footer and choose between:

  1. Contact Form Footer - if you choose this option the email address where you'll receive messages is the one from the Contact Page > Your Email Address field.

  3. Normal 3 Widgets Footer - if you choose this option add widgets (Appearance > Widgets) in one of the locations ( Footer 1, Footer 2, Footer 3)

Theme Options
